This repository documents the interactive Giant Pepper Box project. The Pepper Boxes are in the Arduino Project Hub catagory "Audio and Sound". This repository includes the Arduino progams for Pepper Boxes A, B, C. Also the scrips for 32 of the sound clips as well as the MP3 clips themselves. 

Also in this repository is a draft of the write-up of this project for the Arduino Project Hub; "Pepper Draft" and a photo
